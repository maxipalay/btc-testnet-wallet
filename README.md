# btc-testnet-wallet

Programación de una wallet de bitcoin para TestNet.

Entrega para Blockchain - 2020

### Como usar

- Situado en la carpeta principal ejecutar node index.js
- Se mostrará un menú con numeros
- Para seleccionar la opción se debe ingresar el número, seguido de "Enter"
